gateway receives request
    [+ has csrf request attribute +]
    [+ has access to csrf cookie +]
routes to resource-service
    [- no csrf request attribute -]
    [+ has access to csrf cookie +]
resource-service makes remote call to elasticsearch-service
    [- no csrf request attribute -]
    [- no access to csrf cookie -]









user is authenticated
gateway receies GET request
    + session id (header)
    - csrf (header)
    1. validates session id
    2. forwards request to service via zuul proxy
resource-service receives GET request
    1. makes remote call to elasticsearch-service
    2. returns result
